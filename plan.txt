# Manage Easy - Web UI Implementation Plan (React + MUI)

## Goal
Build a modern web UI for the task management system using React and Material-UI that allows users to:
- Sign in with Google (already enabled)
- View and manage their Ideas, Features, Tasks, and Bugs
- Get their Firebase token for MCP integration

## Technology Stack
- **Frontend:** React 18 with Vite
- **UI Library:** Material-UI (MUI) v5
- **State Management:** React Context + Hooks
- **Firebase SDK:** Firebase JS SDK v10
- **Routing:** React Router v6
- **Build Tool:** Vite
- **Hosting:** GitHub Pages

## UI Design System - Minimalist & Modern

### Design Principles
1. **Minimalism:** Clean interfaces, ample whitespace, focused content
2. **Clarity:** Clear hierarchy, readable typography, intuitive navigation
3. **Consistency:** Unified spacing, predictable interactions, coherent visual language
4. **Responsiveness:** Mobile-first approach, adaptive layouts, touch-friendly targets
5. **Performance:** Smooth animations, fast loading, optimized interactions

### Component Design Guidelines

#### Cards
- Background: Surface color
- Border: 1px solid Border color
- Border Radius: md (8px)
- Padding: md to lg (16-24px)
- Shadow: Low elevation on hover
- Spacing: md (16px) between cards

#### Buttons
- Primary: Filled with primary color, white text
- Secondary: Outlined with primary color
- Text: No background, primary color text
- Height: 40px (md), 36px (sm), 32px (xs)
- Padding: 16px horizontal (md), 12px (sm)
- Border Radius: md (8px)
- Font Weight: 500
- Touch Target: Minimum 44x44px on mobile

#### Forms
- Input Height: 40px
- Border: 1px solid Border color
- Border Radius: sm (4px)
- Focus: 2px primary color ring
- Label: Text Secondary, 14px, above input
- Helper Text: Caption size, Text Tertiary
- Error State: Error color border + text

#### Navigation
- AppBar Height: 64px desktop, 56px mobile
- Drawer Width: 280px desktop, 100% mobile (temporary)
- Active Item: Primary color background (10% opacity) + primary text
- Hover: Surface color change
- Icons: 24x24px

#### Spacing & Layout
- Page Padding: xl (32px) desktop, md (16px) mobile
- Section Gap: xl to 2xl (32-48px)
- Content Max Width: 1280px
- Card Grid Gap: md to lg (16-24px)

#### Animations & Transitions
- Duration: 200ms (fast), 300ms (normal), 400ms (slow)
- Easing: cubic-bezier(0.4, 0, 0.2, 1) - smooth
- Hover: 200ms
- Modal/Dialog: 300ms
- Page Transition: 400ms

### Responsive Design Rules

#### Mobile (< 640px)
- Single column layout
- Full-width cards
- Bottom navigation or hamburger menu
- Drawer slides from left (temporary)
- Stack form fields vertically
- Hide secondary information
- Larger touch targets (minimum 44x44px)
- Reduce padding/margins (16px → 12px)

#### Tablet (640px - 1024px)
- 2-column grid for cards
- Side drawer (persistent or temporary)
- Show more information
- Standard padding/margins

#### Desktop (> 1024px)
- 3-4 column grid for cards
- Persistent side drawer
- Hover states enabled
- Full information visible
- Generous spacing

## API Integration

### Service Layer (src/services/api.js):
```javascript
const BASE_URL = 'https://us-central1-manage-easy-1768423759.cloudfunctions.net';

class ApiService {
  constructor() {
    this.token = null;
  }

  setToken(token) {
    this.token = token;
  }

  async request(endpoint, options = {}) {
    const response = await fetch(`${BASE_URL}/${endpoint}`, {
      ...options,
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${this.token}`,
        ...options.headers,
      },
    });

    if (!response.ok) {
      const error = await response.json();
      throw new Error(error.error || 'Request failed');
    }

    return response.json();
  }

  // Ideas
  async listIdeas() {
    return this.request('listIdeas');
  }

  async createIdea(data) {
    return this.request('createIdea', {
      method: 'POST',
      body: JSON.stringify(data),
    });
  }

  // ... more methods
}

export default new ApiService();
```

### GitHub Pages Deployment:

#### Option 1: Using gh-pages package (Recommended)
1. Install gh-pages:
```bash
npm install --save-dev gh-pages
```

2. Add to package.json scripts:
```json
{
  "scripts": {
    "predeploy": "npm run build",
    "deploy": "gh-pages -d dist"
  },
  "homepage": "https://yourusername.github.io/manage-easy"
}
```

3. Deploy:
```bash
npm run deploy
```

#### Option 2: GitHub Actions (CI/CD)
Create `.github/workflows/deploy.yml`:
```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches: [ main ]
  workflow_dispatch:

permissions:
  contents: read
  pages: write
  id-token: write

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: 18
      - name: Install dependencies
        run: cd app && npm ci
      - name: Build
        run: cd app && npm run build
      - name: Upload artifact
        uses: actions/upload-pages-artifact@v2
        with:
          path: ./app/dist

  deploy:
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    runs-on: ubuntu-latest
    needs: build
    steps:
      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v2
```

#### Vite Configuration for GitHub Pages
Update `vite.config.js` to set the correct base path:
```javascript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
  base: '/manage-easy/', // Replace with your repo name

```

#### SPA Routing on GitHub Pages
Create `public/404.html` to handle client-side routing:
```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Redirecting...</title>
    <script>
      // Redirect to index.html with path in query string
      sessionStorage.redirect = location.href;
    </script>
    <meta http-equiv="refresh" content="0;URL='/'">
  </head>
  <body>
  </body>
</html>
```

Add to `index.html` (in `<head>` before other scripts):
```html
<script>
  // Check for redirect from 404.html
  (function(){
    var redirect = sessionStorage.redirect;
    delete sessionStorage.redirect;
    if (redirect && redirect != location.href) {
      history.replaceState(null, null, redirect);
    }
  })();
</script>
```
